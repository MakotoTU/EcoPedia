# EcoPedia 🌿

EcoPedia adalah aplikasi Android yang dirancang untuk membantu pengguna memahami berbagai jenis sampah dan memberikan tips hidup ramah lingkungan. Aplikasi ini bertujuan untuk meningkatkan kesadaran akan pengelolaan sampah yang benar dan mempromosikan gaya hidup berkelanjutan.

## ✨ Fitur Utama

- **Edukasi Kategori Sampah**: Informasi mendalam tentang berbagai jenis sampah:
  - 🧴 Plastik
  - 📄 Kertas
  - 🍾 Kaca
  - 🍎 Organik
  - 🔋 B3 (Bahan Berbahaya dan Beracun)
  - ⚙️ Logam
- **Tips Ramah Lingkungan**: Berbagai tips praktis untuk mengurangi jejak karbon dan hidup lebih hijau.
- **Mode Gelap/Terang**: Dukungan penuh untuk tema gelap dan terang guna kenyamanan visual.
- **Antarmuka Modern**: Desain bersih dan responsif menggunakan Material Design 3 dan dukungan *edge-to-edge*.

## 🛠️ Tech Stack

- **Language**: Kotlin
- **Platform**: Android (Min SDK 24, Target SDK 36)
- **UI Framework**: XML Layouts with ConstraintLayout
- **Design System**: Material Design 3
- **Dependency Management**: Gradle Version Catalog (libs.versions.toml)

## 🚀 Persyaratan Instalasi

- Android Studio Ladybug atau versi terbaru.
- JDK 11 atau lebih tinggi.
- Perangkat Android dengan API Level minimal 24 (Nougat).

## 📂 Struktur Proyek

```text
app/src/main/java/com/makoto/ecopedia/
├── MainActivity.kt    # Layar utama dengan kategori sampah
├── DetailActivity.kt  # Layar detail untuk tiap kategori
├── TipsActivity.kt    # Layar berisi tips ramah lingkungan
└── SplashActivity.kt  # Layar pembuka (Splash Screen)
```
